# MCP Endpoint

The MCP (Model Context Protocol) endpoint is the primary interface for AI agents to interact with Fizzy. It implements the MCP Streamable HTTP transport specification, enabling AI assistants like Claude, GPT-4, and custom agents to manage tasks in Fizzy.

<Aside type="note">
  MCP is an open standard for AI-to-tool communication. Learn more at
  [modelcontextprotocol.io](https://modelcontextprotocol.io).
</Aside>

## POST /mcp <Badge text="Auth Required" variant="caution" />

The main MCP endpoint for sending JSON-RPC requests. Supports both synchronous responses and SSE streaming.

### Request Headers

| Header                 | Required | Description                               |
| ---------------------- | -------- | ----------------------------------------- |
| `Content-Type`         | Yes      | Must be `application/json`                |
| `Accept`               | Yes      | `application/json` or `text/event-stream` |
| `X-Fizzy-Token`        | Yes      | Your Fizzy Personal Access Token          |
| `X-Fizzy-Account-Slug` | No       | Target account (e.g., `/my-org`)          |
| `X-API-Key`            | No       | Alternative API key authentication        |

### Request Body

Standard JSON-RPC 2.0 request format:

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "tools/call",
  "params": {
    "name": "fizzy_list_boards",
    "arguments": {}
  }
}
```

### Response

JSON-RPC 2.0 response:

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "[{\"id\":\"brd_abc123\",\"name\":\"Product Roadmap\",\"cardCount\":42}]"
      }
    ]
  }
}
```

### Streaming Response (SSE)

When `Accept: text/event-stream` is specified, the response is sent as Server-Sent Events:

```http
HTTP/1.1 200 OK
Content-Type: text/event-stream
Cache-Control: no-cache
Connection: keep-alive

data: {"jsonrpc":"2.0","id":1,"result":{"content":[{"type":"text","text":"..."}]}}

```

## MCP Protocol Flow

<Steps>

1. **Initialize**

   Send an `initialize` request to establish the session and receive server capabilities.

2. **List Tools**

   Send `tools/list` to discover available tools and their schemas.

3. **Call Tools**

   Use `tools/call` with the tool name and arguments to perform operations.

4. **Handle Responses**

   Process the JSON-RPC response containing the tool result or error.

</Steps>

```javascript
async function callTool(name, args = {}) {
  const response = await fetch(MCP_URL, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Accept': 'application/json',
      'X-Fizzy-Token': TOKEN
    },
    body: JSON.stringify({
      jsonrpc: '2.0',
      id: Date.now(),
      method: 'tools/call',
      params: {
        name,
        arguments: args
      }
    })
  });
  
  const { result, error } = await response.json();
  
  if (error) {
    throw new Error(`MCP Error: ${error.message}`);
  }
  
  // Parse the text content
  const textContent = result.content.find(c => c.type === 'text');
  return JSON.parse(textContent.text);
}

// List all boards
const boards = await callTool('fizzy_list_boards');
console.log('Boards:', boards);

````

  <TabItem label="Python">
```python
import json

def call_tool(name, args=None):
    response = requests.post(
        MCP_URL,
        headers={
            'Content-Type': 'application/json',
            'Accept': 'application/json',
            'X-Fizzy-Token': TOKEN
        },
        json={
            'jsonrpc': '2.0',
            'id': 1,
            'method': 'tools/call',
            'params': {
                'name': name,
                'arguments': args or {}
            }
        }
    )

    data = response.json()

    if 'error' in data:
        raise Exception(f"MCP Error: {data['error']['message']}")

    # Parse text content
    text_content = next(
        c for c in data['result']['content']
        if c['type'] == 'text'
    )
    return json.loads(text_content['text'])

# List all boards
boards = call_tool('fizzy_list_boards')
print('Boards:', boards)
````

## Error Handling

MCP errors are returned as JSON-RPC error responses:

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "error": {
    "code": -32600,
    "message": "Invalid Request",
    "data": {
      "details": "Missing required parameter: board_id"
    }
  }
}
```

### Error Codes

| Code     | Name             | Description               |
| -------- | ---------------- | ------------------------- |
| `-32700` | Parse Error      | Invalid JSON              |
| `-32600` | Invalid Request  | Invalid JSON-RPC request  |
| `-32601` | Method Not Found | Unknown method            |
| `-32602` | Invalid Params   | Invalid method parameters |
| `-32603` | Internal Error   | Server error              |

### Tool Errors

When a tool fails, it returns an error in the result content:

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "content": [
      {
        "type": "text",
        "text": "Error: Card not found\nCode: NOT_FOUND_ERROR"
      }
    ],
    "isError": true
  }
}
```

```javascript
async function callToolWithStreaming(name, args = {}) {
  const response = await fetch(MCP_URL, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Accept': 'text/event-stream',
      'X-Fizzy-Token': TOKEN
    },
    body: JSON.stringify({
      jsonrpc: '2.0',
      id: Date.now(),
      method: 'tools/call',
      params: { name, arguments: args }
    })
  });
  
  const reader = response.body.getReader();
  const decoder = new TextDecoder();
  
  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    
    const chunk = decoder.decode(value);
    const lines = chunk.split('\n');
    
    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const data = JSON.parse(line.slice(6));
        console.log('Received:', data);
      }
    }
  }
}
```

## Complete Example

Here's a complete example that initializes, lists boards, and creates a card:

```javascript
const TOKEN = 'fz_xxxxxxxxxxxxx';
const MCP_URL = 'https://fizzy-mcp-hosted.ryanyogan.workers.dev/mcp';

async function mcpRequest(method, params = {}) {
  const response = await fetch(MCP_URL, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      Accept: 'application/json',
      'X-Fizzy-Token': TOKEN,
    },
    body: JSON.stringify({
      jsonrpc: '2.0',
      id: Date.now(),
      method,
      params,
    }),
  });

  const data = await response.json();
  if (data.error) throw new Error(data.error.message);
  return data.result;
}

async function callTool(name, args = {}) {
  const result = await mcpRequest('tools/call', { name, arguments: args });
  const text = result.content.find((c) => c.type === 'text')?.text;
  return text ? JSON.parse(text) : null;
}

// Main workflow
async function main() {
  // 1. Initialize
  console.log('Initializing MCP session...');
  const init = await mcpRequest('initialize', {
    protocolVersion: '2024-11-05',
    capabilities: {},
    clientInfo: { name: 'my-agent', version: '1.0.0' },
  });
  console.log(`Connected to ${init.serverInfo.name} v${init.serverInfo.version}\n`);

  // 2. Get identity
  const identity = await callTool('fizzy_get_identity');
  console.log(`Logged in as: ${identity.user.name}`);
  console.log(`Accounts: ${identity.accounts.map((a) => a.name).join(', ')}\n`);

  // 3. List boards
  const boards = await callTool('fizzy_list_boards');
  console.log('Boards:');
  boards.forEach((b) => console.log(`  - ${b.name} (${b.cardCount} cards)`));

  // 4. Create a card on the first board
  if (boards.length > 0) {
    const card = await callTool('fizzy_create_card', {
      board_id: boards[0].id,
      title: 'Task from AI Agent',
      description: 'This card was created via the MCP endpoint.',
    });
    console.log(`\nCreated card #${card.number}: ${card.title}`);
  }
}

main().catch(console.error);
```

### Custom Agents

Use the MCP SDK for your language:

```bash
npm install @modelcontextprotocol/sdk
```

```javascript
import { Client } from '@modelcontextprotocol/sdk/client/index.js';
import { StreamableHTTPClientTransport } from '@modelcontextprotocol/sdk/client/streamableHttp.js';

const transport = new StreamableHTTPClientTransport(
  new URL('https://fizzy-mcp-hosted.ryanyogan.workers.dev/mcp'),
  {
    requestInit: {
      headers: {
        'X-Fizzy-Token': 'fz_xxxxxxxxxxxxx',
      },
    },
  },
);

const client = new Client({ name: 'my-agent', version: '1.0.0' });
await client.connect(transport);

const tools = await client.listTools();
console.log(
  'Available tools:',
  tools.tools.map((t) => t.name),
);
```
