## What is Fizzy MCP?

Fizzy MCP is a **Model Context Protocol (MCP)** server that connects AI assistants like Claude, Cursor, and OpenCode to [Fizzy](https://app.fizzy.do) - the AI-powered task management platform.

With Fizzy MCP, your AI can:

- Create, update, and manage cards and boards
- Triage work into workflow columns
- Assign tasks to team members
- Track progress with tags and golden cards
- Collaborate through comments

## Deployment Modes

Fizzy MCP offers two deployment modes to fit your workflow:

<CardGrid>
  <Card title="Remote Mode (Recommended)" icon="cloud">
    Use our hosted service at `fizzy-mcp-hosted.ryanyogan.workers.dev`. No local server required -
    just configure and go. **Best for:** Quick setup, remote MCP clients, environments where local
    installation isn't practical. **Rate limits:** 100 writes/day (anonymous) or 1,000/day (with API
    key)
  </Card>
  <Card title="Local Mode" icon="laptop">
    Run the MCP server locally via stdio transport. Full control with no rate limits. **Best for:**
    Desktop AI assistants, development, testing, and offline usage.
  </Card>
</CardGrid>

## Quick Install

```bash
npx fizzy-do-mcp auth
```

This will:

1. Ask you to choose **Remote** (recommended) or **Local** mode
2. Open your browser to get a Fizzy Personal Access Token
3. Optionally configure an API key for higher rate limits (remote mode)
4. Auto-configure detected AI agents (Claude Desktop, Cursor, OpenCode)

## Architecture

### Remote Mode (Hosted Service)

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   AI Agent      │────▶│  Fizzy MCP      │────▶│   Fizzy API     │
│ (Claude, etc.)  │◀────│  Hosted Service │◀────│ (app.fizzy.do)  │
└─────────────────┘     └─────────────────┘     └─────────────────┘
        │                       │
        │                       │
        ▼                       ▼
   MCP over HTTP          Cloudflare Workers
   (Streamable)           (D1 + KV)
```

### Local Mode (Stdio)

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   AI Agent      │────▶│   Fizzy MCP     │────▶│   Fizzy API     │
│ (Claude, etc.)  │◀────│  Local Server   │◀────│ (app.fizzy.do)  │
└─────────────────┘     └─────────────────┘     └─────────────────┘
        │                       │
        │                       │
        ▼                       ▼
   MCP Protocol          TypeScript Client
   (JSON-RPC stdio)      (Result types)
```

## Hosted Service

The hosted service at `https://fizzy-mcp-hosted.ryanyogan.workers.dev` provides:

- **Remote MCP Access** - Use Fizzy tools from any MCP client via HTTP
- **Rate Limiting** - Tiered limits based on authentication
- **Usage Tracking** - Monitor your API usage
- **Zero Setup** - No local server required
