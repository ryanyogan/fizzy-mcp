# Introduction to Fizzy MCP

Fizzy MCP is a **Model Context Protocol (MCP)** server that enables AI assistants to interact with [Fizzy](https://app.fizzy.do), the AI-powered task management platform built for modern teams.

## What is MCP?

The [Model Context Protocol](https://modelcontextprotocol.io) is an open standard developed by Anthropic that allows AI applications to securely connect to external data sources and tools. It provides a standardized way for AI assistants to:

- **Access data** from external systems
- **Execute actions** through well-defined tools
- **Maintain context** across conversations

## Deployment Modes

Fizzy MCP offers **two deployment modes** to fit different use cases:

<Tabs>
  <TabItem label="Remote Mode (Recommended)">
    Use our hosted MCP service running on Cloudflare Workers. Your AI agent connects via HTTP to `https://fizzy-mcp-hosted.ryanyogan.workers.dev/mcp`.

    **Advantages:**
    - No local server to run or maintain
    - Works with any MCP client that supports HTTP transport
    - Automatic updates - always running the latest version
    - Perfect for remote development environments

    **Considerations:**
    - Rate limits apply (100 writes/day anonymous, 1,000/day with API key)
    - Requires internet connection
    - Your Fizzy token is sent to the hosted service

    ```json
    // Example Claude Desktop config (Remote Mode)
    {
      "mcpServers": {
        "fizzy": {
          "transport": "http",
          "url": "https://fizzy-mcp-hosted.ryanyogan.workers.dev/mcp",
          "headers": {
            "X-Fizzy-Token": "your-fizzy-token"
          }
        }
      }
    }
    ```

  </TabItem>
  <TabItem label="Local Mode">
    Run the MCP server locally on your machine. Your AI agent connects via stdio (standard input/output).

    **Advantages:**
    - No rate limits
    - Works offline (once authenticated)
    - Token stays on your machine
    - Full control over the server

    **Considerations:**
    - Requires Node.js 20+
    - Server must be installed and available
    - May need manual updates

    ```json
    // Example Claude Desktop config (Local Mode)
    {
      "mcpServers": {
        "fizzy": {
          "command": "npx",
          "args": ["-y", "fizzy-do-mcp@latest"]
        }
      }
    }
    ```

  </TabItem>
</Tabs>

## Features

### 40 MCP Tools

Fizzy MCP provides comprehensive API coverage across 7 categories:

| Category | Tools | Description                       |
| -------- | ----- | --------------------------------- |
| Identity | 2     | Account discovery and settings    |
| Boards   | 7     | Board CRUD and publishing         |
| Cards    | 18    | Card lifecycle, assignments, tags |
| Comments | 5     | Discussion threads on cards       |
| Columns  | 5     | Workflow stage management         |
| Tags     | 1     | Label management                  |
| Users    | 2     | Team member information           |

### Auto-Configuration

After authentication, the CLI can automatically detect and configure:

- **Claude Desktop** - Anthropic's official desktop app
- **Cursor** - AI-powered code editor
- **OpenCode** - Open-source AI coding assistant

The CLI generates the correct configuration for your chosen mode (remote or local) and writes it directly to the agent's config file.

### CLI Commands

Fizzy MCP includes a full-featured CLI for setup and management:

| Command               | Description                                     |
| --------------------- | ----------------------------------------------- |
| `fizzy-mcp auth`      | Configure authentication (mode, token, API key) |
| `fizzy-mcp configure` | Auto-configure AI agents                        |
| `fizzy-mcp whoami`    | Show current identity and account               |
| `fizzy-mcp status`    | Check configuration status                      |
| `fizzy-mcp logout`    | Clear stored credentials                        |

### Type-Safe Client

Built with TypeScript and Zod validation:

```typescript
import { FizzyClient } from '@fizzy-mcp/client';

const client = new FizzyClient({
  accessToken: 'your-token',
  accountSlug: '/your-org',
});

// Fully typed responses
const result = await client.cards.list({ board_ids: ['abc123'] });
if (result.ok) {
  console.log(`Found ${result.value.length} cards`);
}
```

### Resilient API Calls

Automatic retry logic with exponential backoff:

- Retries on network errors and 5xx responses
- Respects rate limit headers
- Configurable retry count and delays

## Prerequisites

Before getting started, you'll need:

1. **A Fizzy account** - Sign up at [app.fizzy.do](https://app.fizzy.do)
2. **Node.js 20+** - Required for the CLI and local mode
3. **An MCP-compatible AI agent** - Claude Desktop, Cursor, or OpenCode

<Aside type="tip">
  For **remote mode**, you only need Node.js to run the initial `fizzy-mcp auth` command. After
  that, your AI agent connects directly to the hosted service.
</Aside>
